# Tosh Labs Site

A fresh Next.js marketing site for Tosh Labs, the Vancouver-based corporation behind Pulsr and Pausr.

## Stack

- Next.js 14
- React 18
- TypeScript
- CSS via `app/globals.css`

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Current structure

- `app/page.tsx`: homepage
- `components/ProductCard.tsx`: reusable product card
- `app/globals.css`: design system and page styling

## Positioning

- `Tosh Labs`: parent company focused on innovation and software technologies
- `Pulsr`: execution intelligence for engineering managers
- `Pausr`: operational clarity for time-off planning and approvals
